# 101312826_lab_test1_chat_app

## Chat App

A real-time web chat application built with React, Node.js, Express, Socket.io, and MongoDB, supporting both group chats
and private messaging between users.

## Features

### User Authentication

Signup with a unique username, first name, last name, and password or login with an existing account

### Real-time Chat

Join predefined rooms to chat with other users (e.g., devops, cloud computing, covid19, sports, nodeJS)
Or join private messaging between two users. there is a list of currently connected users for all loggined user to see,
click a user to start a private chat

## How to use
To start using the app, please create an account with an unique username. Then once you sign in on the left sidebar
there will be a list of premade rooms to enter and you can chat with anyone in the room while also viewing the chat 
history before you entered the room. On the right sidebar there will be a list of all active connected users and you 
can click on the to privately message them. All messages are saved to mongoDB so you can always view your old chats
